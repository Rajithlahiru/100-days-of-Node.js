# Introduction

- ##### Extended from JavaScript.
- ##### Java Script Object Notation (JSON).
- #### Can hold

#### Key/Name value pairs
```json
{"name":"value"}
```

#### Objects
```json
{"name1":"value","name2":"value","name3":"value"}
```

#### Arrays
```json
{"name":
    [{
        "name1":"value", 
        "name2":"value", 
        "name3":"value"
        }]
    }
```
